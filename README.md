## Description

  An API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This project 
  uses Express.js for routing, a MongoDB database, the Mongoose ODM and Moment for Date string manipulation.
  
## Installation

Installation is a process of installing the dependencies required.
Intialize node package manager and then run the following commands; 
```script 
npm install
```

## Usage

 In order to use the application, run the following commands:  
```script
npm run seed
npm start
```
